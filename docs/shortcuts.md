# ⌨️ 快捷键与命令面板指南

为了提升操作效率，本系统内置了仿 IDE 式的快捷键支持。

## 全局命令面板

按 `Cmd + K` (Mac) 或 `Ctrl + K` (Win) 呼出。

- **快速搜索**: 支持模糊匹配项目名、分类名或文档名。
- **键盘导航**: 使用 `↑` / `↓` 移动，`Enter` 确认跳转。

## Vim 风格直达组合键

无需呼出面板，连续输入以下字母即可快速切换。

| 组合键  | 功能                            |
| :------ | :------------------------------ |
| `g` `h` | **G**o **H**ome (工作台)        |
| `g` `d` | **G**o **D**ashboard (系统监控) |
| `g` `p` | **G**o **P**rojects (项目空间)  |
| `g` `c` | **G**o **C**onfig (系统配置)    |

## 其他支持

- `Esc`: 在任何状态下关闭弹窗或搜索框。
